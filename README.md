# Cherisha KG — Portfolio

A multidisciplinary portfolio showcasing selected work across software engineering, data analytics, artificial intelligence, machine learning, and product management.

## Featured work

- **Swasthatha** — a privacy-focused mental wellness companion with a detailed product case study
- **Cleft Detection in Antenatal Scans** — computer-vision research with the paper and conference presentation certificate
- **Restaurant Intelligence** — a data analytics and operational decision-making dashboard concept
- **Pet Next Door** — a full-stack community platform for pet owners

## Highlights

- Interactive 3D hero experience
- Responsive project archive organized by discipline
- Product and data case studies
- Research evidence and downloadable resume
- Keyboard-accessible command palette
- Smooth, performance-conscious motion

## Technology

Next.js, React, TypeScript, Tailwind CSS, Framer Motion, Three.js, React Three Fiber, Lenis, and Lucide.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```

## Deploy on Vercel

Import this repository in Vercel and keep the automatically detected Next.js settings. Every push to `main` will trigger a new deployment.
